# 提醒 · 番茄钟 · 快捷倒计时 · 备忘录 —— 组合版块设计

- 分支:`feature/reminder-pomodoro`
- 日期:2026-05-31
- 状态:设计已确认,待转实现计划

## 1. 概述与目标

为「蓝色小嗵」桌宠新增一个围绕"时间管理"的组合版块,四个子功能共享同一套"计时核心 + 浮窗 + 持久气泡 + 独立窗口",避免重复造轮子:

| 子功能 | 心智模型 | 核心价值 |
|---|---|---|
| 快捷倒计时 | 厨房计时器 | 随手数个数,临时、看着它走 |
| 番茄钟 | 节拍器 | 专注/休息自动接龙,结构化专注 |
| 提醒 | 闹钟 / 日程 | 到点叫你,设了就忘、持久 |
| 备忘录 | 便签 | 无计时的纯文本笔记 |

原则:按风险从低到高分阶段交付,每阶段独立可验。约束:仅 Windows(项目既有约定);复用既有视觉语言(StatusPanel / 气泡 / FloatLabel)与持久化方式(`geren/` 便携 JSON)。

## 2. 架构

### 2.1 计时核心(共享地基)

一个纯逻辑(无 Qt、可单测)的倒计时原语:每个计时实例有 id、标签、目标时刻、剩余时间、到 0 回调。提醒、番茄钟、快捷倒计时都是它的消费者,区别只在"到 0 时做什么"和"生命周期"。

### 2.2 模块 / 文件清单

新增(纯逻辑):
- `src/timer_core.py` —— 倒计时原语 `Countdown`(start / remaining / is_done)。
- `src/reminder_system.py` —— 提醒 + 备忘录的 CRUD、持久化、标记解析器、到点检测。
- `src/pomodoro.py` —— 番茄钟状态机(阶段、轮数、时长配置),建在 `Countdown` 上。

新增(Qt 界面):
- `src/countdown_float.py` —— 桌宠头顶浮窗,显示当前前台 / 最近的倒计时。
- `src/reminder_bubble.py` —— 持久、可点击关闭、可堆叠的提醒气泡。
- `src/reminder_window.py` —— 独立窗口:快捷倒计时 / 番茄钟 / 提醒 / 备忘录 控制台。

改动既有:
- `src/user_data.py` —— 新增 `reminder_data_path()` → `geren/reminder_data.json`。
- `main.py` —— 持有 ReminderSystem + Pomodoro + 前台计时器状态;主循环每秒检测到点;接气泡 / 浮窗;托盘加入口;触发桌宠联动动画。
- `src/chat_service.py` —— 系统提示词加一句"识别到提醒意图就输出标记"(聊天路径)。
- `src/status_panel.py` —— 聊天回复处理里扫描标记 → 创建提醒、隐藏标记;注入 `reminder_system`。

### 2.3 三种消费者对照

| | 快捷倒计时 | 番茄钟 | 提醒 |
|---|---|---|---|
| 到 0 时 | 弹持久气泡 | 自动进下一阶段 | 弹持久气泡(点击关) |
| 生命周期 | 临时,关程序即没 | 临时会话(时长配置持久) | 持久,关机重开仍在 |
| 数量 | 同时一个前台 | 同时一个前台 | 可排多个(后台) |
| 占头顶浮窗 | 是(前台) | 是(前台) | 否(后台;仅无前台计时器时显示最近到期) |
| 入口 | 窗口按钮 / 输入 | 窗口开始 | 聊天标记(AI 识别)/ 窗口表单(本地解析) |

前台计时器(番茄钟、快捷倒计时)**同一时刻只跑一个**。

## 3. 数据模型与持久化

文件:`geren/reminder_data.json`(便携,跟随程序)。

```jsonc
{
  "reminders": [{
    "id": "r_<ts>_<rand>",
    "text": "喝水",
    "kind": "relative" | "absolute" | "daily",
    "due_ts": 1717201800.0,      // 下次触发的 epoch 秒
    "clock": "18:00",            // absolute / daily 用(显示 + 顺延)
    "created_ts": 1717200000.0,
    "status": "active" | "done" | "missed",
    "fired_ts": 0.0
  }],
  "memos": [{
    "id": "m_<ts>_<rand>", "text": "...", "created_ts": 0.0, "updated_ts": 0.0
  }],
  "pomodoro": {                  // 仅配置持久化,会话本身不持久
    "focus_min": 25, "short_break_min": 5,
    "long_break_min": 15, "cycles_before_long": 4
  }
}
```

`kind` / `clock` 字段从 P1 起就预留,P4 加绝对 / 每日时不必迁移数据。快捷倒计时、番茄钟会话本身不写盘(临时);仅番茄钟时长配置持久。

## 4. 标记 / 解析

### 4.1 标记格式(内部统一表示)

`[<时间>:<事件>]`,一条消息可含多个。

| 类型 | 阶段 | 例子 |
|---|---|---|
| 相对 | P3 | `[30min:喝水]` `[90s:面条]` `[1h30m:取快递]` `[2小时:开会]` |
| 绝对时钟 | P4 | `[18:00:吃药]` `[9:30:开会]` |
| 每日重复 | P4 | `[每天9:00:吃药]` `[每天22:30:睡觉]` |

### 4.2 双路径

- **聊天输入(自然语言)→ 调用 API**:AI 把人话(如"30 分钟后提醒我喝水")转成标记,程序本地解析标记成提醒,并从展示气泡里隐藏标记,只显示友好回复("好的,30 分钟后提醒你喝水~")。需在系统提示词加入标记格式说明 + 触发条件;回复处理里扫描并剥离标记。
- **窗口直接输入 → 本地解析**:表单填时间 + 事件(或直接打标记),本地规则解析,不调 API,离线确定。
- 两路共用同一个本地"标记 / 结构 → 提醒"解析器。
- **离线退化**:聊天路径在 API 未配置时退回本地标记解析(手打 `[30min:喝水]` 仍生效,自然语言识别不可用)。

## 5. 行为与边界

- **提醒到点(程序运行中)** → 持久气泡:不自动消失,点击关闭;同时到点多个 → 纵向堆叠。
- **快捷倒计时到 0** → 持久气泡(同一个气泡组件);文案 `⏰ 时间到!`(有标签则 `⏰ <标签> 时间到`)。
- **番茄钟阶段切换** → 普通自动淡出气泡播报("专注结束,休息一下吧~" / "休息结束,继续加油!")+ 桌宠动画切换;动画本身是该阶段的持续指示,故气泡不必常驻。
- **提醒错过(程序关闭期间到点)** → 一次性提醒标 `missed`(窗口里标红,不弹气泡);**每日提醒不算错过**,直接顺延到下一个未来时刻。
- **绝对时钟已过当天** → 顺延到明天同一时刻(而非立即错过)。
- **浮窗优先级**:前台计时器(番茄钟 / 快捷倒计时)> 最近到期的提醒 > 隐藏。前台计时器结束后,浮窗回落到"最近到期的提醒",没有则隐藏。
- **桌宠联动**:番茄钟专注阶段 → `study`(ONESHOT_STAY,定格埋头);休息阶段 → `wake`(ONESHOT_RETURN,舒展后回自由 idle)。`study` 定格会被任意互动打断,不强锁。

## 6. 界面

- **头顶浮窗**(`countdown_float.py`):贴桌宠头顶,随桌宠移动(复用 `bubble.update_position` 思路);显示 `标签 MM:SS`(>1h 用 H:MM:SS)。聊天气泡与浮窗同时出现时,气泡上移避让。
- **持久气泡**(`reminder_bubble.py`):类聊天气泡,但去掉 `WindowTransparentForInput`(可点)+ 去掉淡出定时器(不自动消失);显示 `⏰ 事件` + 关闭提示;支持多个堆叠。由"提醒到点"和"快捷倒计时到 0"共用。
- **独立窗口**(`reminder_window.py`):无边框 `Tool` 窗,沿用既有面板风格与 DPI 缩放;托盘新增入口「⏰ 提醒·番茄钟」。分区:快捷倒计时(预设 5/10/15/25 + 自定义 + 可选标签)、番茄钟(开始/暂停/重置 + 时长设置)、提醒(列表:进行中/已错过/已完成,增删改)、备忘录(便签增删改)。随阶段逐步长出。

## 7. 分阶段实施计划

每阶段独立可验,逐个确认后再做下一个。

### P1 —— 地基 + 快捷倒计时
计时核心、头顶浮窗、持久气泡、最小独立窗口(先只放快捷倒计时:预设按钮 + 自定义输入 + 可选标签)。
- **验证**:窗口点"5 分钟"→ 头顶倒数 → 到 0 弹气泡,点击关闭。

### P2 —— 番茄钟
窗口加番茄钟分区(开始/暂停/重置 + 时长设置);状态机自动接龙;阶段切换弹气泡;桌宠联动(专注 = `study`,休息 = `wake`)。
- **验证**:开始 → 专注/休息自动切换 → 切换时弹气泡;专注时桌宠埋头、休息时舒展。

### P3 —— 提醒(相对时间)
本地标记解析器(相对);聊天路径(系统提示词 + AI 生成标记 + 回复扫描隐藏 + 离线退化);窗口提醒列表(增删改);持久化;关机错过标 `missed`。
- **验证**:聊天说"30 分钟后提醒我喝水"→ 后台倒计时 → 到点弹持久气泡;窗口手填一条;关机重开看错过标记。

### P4 —— 提醒进阶
解析器与 AI 指令扩展到绝对时钟 `[18:00:…]` 与每日重复 `[每天9:00:…]`;每日到点自动顺延;窗口显示"每日"标记。
- **验证**:设绝对 / 每日提醒,确认触发与每日顺延。

### P5 —— 备忘录
窗口加备忘录分区(纯文本便签,增删改,持久化)。
- **验证**:增删改便签,重开仍在。

## 8. 待实现时确认的小决策(已给默认值)

1. 前台计时器互斥:番茄钟与快捷倒计时同一时刻只跑一个,开新的会提示 / 替换。
2. 浮窗显示最近到期的提醒时,默认"不论多久都显示";如嫌占屏,可加"仅 X 分钟内才显示"阈值。
3. 备忘录为纯无计时笔记,不与提醒互转。
4. 番茄钟完成暂不计入既有游戏系统(学习任务 / 成就);后续可加。
5. 托盘入口名暂定「⏰ 提醒·番茄钟」。
